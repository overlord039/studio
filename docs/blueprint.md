# **App Name**: HousieHub

## Core Features:

- User Authentication: Allow users to register and log in to their accounts to track their game statistics and access personalized features.
- Lobby Management: Enable users to create and join multiplayer Housie game rooms with customizable settings, such as ticket price, lobby size, and winning format.
- Ticket Generation: Automatically generate Housie tickets following a 3x9 grid format with numbers in specific ranges and randomized positions.
- Number Calling System: Randomly select and display numbers from 1 to 90, with a visual representation of the called numbers and an option to minimize the number board.
- Number Marking Assistance: Visually differentiate between called, marked, and unmarked numbers on the player's ticket with clear indicators.
- Automated Prize Claim Validation: Validate player claims for winning patterns, such as Jaldi 5, Top Line, Middle Line, Bottom Line, and Full House, ensuring only the first valid claim is accepted.
- Voice Announcements: Tool for voice announcement to call the drawn numbers for the convenience of players.

## Style Guidelines:

- Primary color: Deep Indigo (#663399) for a sense of tradition and excitement.
- Background color: Light lavender (#F0F0FF) to provide a gentle contrast to the bold primary.
- Accent color: Electric Purple (#8842B5) for call-to-action buttons and significant elements, drawing the user's attention.
- Use clear and legible sans-serif fonts for game interface elements and headings, optimized for both desktop and mobile screens.
- Employ clean and simple icons that reflect the lighthearted spirit of Housie.
- Maintain a balanced layout with a clear division of sections, ensuring interactive components are easily accessible on both desktop and mobile interfaces.
- Use smooth and subtle transition animations for revealing numbers and validating claims to provide clear and immediate feedback to user actions.
